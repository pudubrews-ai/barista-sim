# Cafe Sim: Presented by Pudubrews

A barista simulator game where you manage a coffee shop, hire staff, buy supplies, and serve customers.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/pudubrews-ai/barista-sim.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd barista-sim
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

## Playing the Game

1.  Start the development server:
    ```bash
    npm run dev
    ```
2.  Open your browser to `http://localhost:5173`.
3.  Click **Start Game**.

### How to Play

*   **Manage Staff**: Hire baristas to serve customers. Watch the queue!
*   **Buy Supplies**: Keep an eye on your Beans, Milk, and Cups. If you run out, customers will leave angry.
*   **Set Prices**: Adjust prices to maximize profit, but don't make them too high or customers won't buy.
*   **Watch the Queue**: If the queue gets too long (red), hire more staff or speed up service.

## Features

*   **Economy**: Realistic costs and wages.
*   **Queue Dynamics**: Rush hours and service times.
*   **Dark Mode**: Sleek "Beantagger" style interface.
